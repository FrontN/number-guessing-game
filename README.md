Number Guessing Game 🎯A sleek, functional command-line interface (CLI) game built with Python. Test your intuition by guessing a randomly generated number within a specific range based on your chosen difficulty.🚀 FeaturesDifficulty Levels: Choose between Easy (numbers 1-10, 5 attempts) and Hard (numbers 1-100, 3 attempts).Input Validation: Handles non-numeric inputs and out-of-range guesses gracefully without crashing.Smart Memory: Tracks your previous guesses and prevents you from losing attempts by repeating the same number.Win Streak System: Keeps track of your current consecutive wins.Clean UI: Features automated screen clearing and loading animations for a better terminal experience.🛠️ Technical HighlightsModular Design: Functions are separated by responsibility (clear_screen, get_game_settings, play_round, etc.).Efficient Lookups: Uses a Python set() for tracking attempted numbers, ensuring $O(1)$ time complexity for duplicate checks.Standard Entry Point: Implements the if __name__ == "__main__": idiom for professional script execution and modularity.📋 PrerequisitesPython 3.x installed on your system.A file named number_guessing_art.py in the same directory containing a logo variable (ASCII art).🕹️ How to PlayClone or Download the repository.Open your terminal or command prompt.Navigate to the project folder and run:Bashpython main.py
